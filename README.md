# take_text_out_inline

## What is this Script

InDesign CS5で、インラインテキストをフレームの外に追い出して指定のタグで囲みます。インライングラフィックはファイル名をタグで囲みます。

検索対象はドキュメントのみ。グループ化されたものは処理しない。

- 2013-09-09  インライングラフィックの場合、ファイル名を挿入するようにした。
